# Alarm-Clock
A simple alarm clock using HTML, CSS, and JavaScript with real-time clock display and alarm functionality.

⏰ Alarm Clock Web App

📌 Overview

This is a simple Alarm Clock web application built using HTML, CSS, and JavaScript. It displays the current time in real-time and allows users to set an alarm. When the set time is reached, an alarm sound is played.


🚀 Features

* Live digital clock (updates every second)
* Set alarm using time input
* Audio alert when alarm triggers
* Clean and minimal user interface


🛠️ Technologies Used

* HTML – Structure of the application
* CSS – Styling and layout
* JavaScript – Functionality and logic

Project Structure
  alarm-clock/
│── index.html     # Main HTML file
│── style.css      # Styling file
│── script.js      # JavaScript logic


▶️ How to Run

1. Download or copy all files into one folder
2. Make sure file names are:
    * index.html
    * style.css
    * script.js
3. Open index.html in any web browser


⏱️ How It Works

* The clock updates every second using setInterval()
* User selects a time using the input field
* When current time matches alarm time:
    * Alarm sound plays
    * Status message updates


🔧 Future Enhancements

* Snooze button
* Stop alarm button
* Multiple alarms support
* Save alarms using local storage
* Dark/light mode toggle


⚠️ Notes

* Alarm works only when the browser tab is open
* Time format depends on system locale
